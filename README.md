# Password Generator

A modern, secure password generator application built with React. Create strong, customizable passwords with an intuitive and visually appealing interface.

## Features

✅ Configurable password length (8-32 characters)
✅ Character type options: uppercase, lowercase, numbers, symbols
✅ One-click password generation
✅ Copy to clipboard functionality
✅ Modern gradient UI design
✅ Responsive layout for all devices
✅ No external dependencies beyond React

## Installation

```bash
# Install dependencies
npm install

# Start development server
npm start

# Build for production
npm run build
```

## Usage

1. Adjust password length using the slider
2. Select which character types to include
3. Click "Generate Password"
4. Click "Copy to Clipboard" to use the generated password

## Technologies

- React 19
- JavaScript
- Modern CSS3 with gradients and animations
- Browser Clipboard API
